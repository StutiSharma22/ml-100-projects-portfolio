# ml-100-projects-portfolio
A comprehensive, hands-on portfolio of 100 machine learning projects designed to build strong intuition across data understanding, feature engineering, modeling, evaluation, explainability, and production thinking.  

# Repository Structure
This is a meta-repository that organizes 100 projects into 4 logical phases.
``` ml-100-projects-portfolio/
├── ml-foundations-lab/          # Projects 1–25
├── classical-ml-lab/            # Projects 26–50
├── advanced-ml-lab/             # Projects 51–75
├── genai-ml-systems-lab/        # Projects 76–100
└── README.md
```

# Learning Phases
## Phase 1: ML Foundations (Projects 1–25)
#### Repository: ml-foundations-lab 
Link- [ml_foundations_lab] ml_foundations_lab

Topics covered:
- Exploratory Data Analysis (EDA)
- Missing values & outliers
- Feature engineering & selection
- Model evaluation & validation
- Bias–variance tradeoff
- Class imbalance & cost-sensitive learning
- Explainability & reproducibility

## Phase 2: Classical Machine Learning (Projects 26–50)
#### Repository: classical-ml-lab
Topics covered:
- Linear & logistic regression
- Regularization techniques
- Decision trees & ensembles
- Random Forest, XGBoost, LightGBM
- Support Vector Machines
- KNN & distance-based learning
- Ensemble methods & stacking

## Phase 3: Advanced & Applied ML (Projects 51–75)
#### Repository: advanced-ml-lab
Topics covered:
- Time series forecasting
- Recommendation systems
- Anomaly & fraud detection
- Natural Language Processing (NLP)
- Text classification & embeddings
- Model monitoring & data drift

## Phase 4: GenAI & ML Systems (Projects 76–100)
#### Repository: genai-ml-systems-lab
Topics covered:
- Large Language Models (LLMs)
- Prompt engineering
- Sentiment analysis using LLMs
- Retrieval-Augmented Generation (RAG)
- Vector databases
- AI application deployment
- Responsible & safe AI systems
